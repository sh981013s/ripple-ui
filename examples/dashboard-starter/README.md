# Dashboard starter built with calmo-ui

This example shows how to use `calmo-ui` for an operations dashboard with:

- `TopBar`
- `Card`
- `SectionHeader`
- `BarChart`
- `List`
- `ListRow`

Use it when you need a React dashboard component library for product surfaces, admin tools, and structured internal dashboards.

## Install

```bash
npm install calmo-ui
```
