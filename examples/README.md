# Examples built with calmo-ui

These example folders are intended to improve install-to-output trust for visitors arriving from search, npm, GitHub, and AI tools.

## Available examples

- [dashboard-starter](./dashboard-starter) — analytics dashboards, cards, list rows, and compact product summaries
- [settings-account-app](./settings-account-app) — account preferences, workspace settings, and calm form surfaces
- [finance-sheet-flow](./finance-sheet-flow) — bottom sheets, result states, dialogs, and payment-adjacent product flows

Each example is labeled `Built with calmo-ui` and is designed to be copied into real projects.
